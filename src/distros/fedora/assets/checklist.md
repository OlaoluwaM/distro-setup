# Backup Checklist

You should ensure that all of the following have been backed up before moving to another OS

- Projects
- Files (with Deja Dup)
- Dconf settings
  - Deja Dup settings
- Wallpapers
- Fonts
- icons
  - Colloid Icon
- Themes
  - Catppuccin-Mocha-Lavender
- Installed RPM Packages
- Installed Rust Crates
- Github CLI Extensions
- Github CLI Aliases
- DNF Aliases
- Global NPM Packages
